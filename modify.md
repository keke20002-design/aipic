🔥 1. “분석하는 척” 연출 (필수)

실제로는 API 기다리는 거지만
👉 유저한테는 AI가 열일하는 느낌 줘야 한다

예:

🧠 AI가 방 구조 분석 중...
📦 물건 밀집도 계산 중...
🧹 청결 상태 스캔 중...
😶 팩폭 문장 생성 중...

➡ 1~2초마다 바뀌게
➡ 핵심: “지금 뭐 하는지 보여주기”

🎯 2. 진행률 가짜라도 만들어

진짜 퍼센트 없어도 된다. 다들 속는다.

예:

12% → 37% → 68% → 91%

➡ 갑자기 100% 가면 됨
➡ 사람은 숫자 보면 안심한다 (이상하게도)

🎮 3. 이미지 위에 스캔 애니메이션

이거 하나 넣으면 퀄리티 급상승

👉 추천:

위에서 아래로 빛줄기 스캔
얼굴 인식 박스처럼 영역 표시
blur → 선명해지는 효과

➡ “AI 같다” 느낌 강하게 남음
